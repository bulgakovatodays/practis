##СТРОКА

String("название переменной") - превращает число в строку, а Number("название строки") - наоборот.

строка.includes(слово) - проверяет есть ли совпадение (результат - true/false)

строка.replace(s1, s2) - Заменяет в строке первое  "s1" на "s2" и выводит результат

строка.replaceAll(s1, s2)  =  строка.replace(/s1/g, "s2"); Заменяет в строке все  "s1" на "s2" и выводит результат

строка.split(separator) - Разделяет строку на массив подстрок используя separator

строка.startsWith(слово, индекс) - проверяет начинается ли строка с этого слова и может отсчитывать символы 

строка.endsWith(слово) - проверяет заканчивается ли строка на это слово

строка.toLowerCase() - для перевода в нижний регистр

строка.toUpperCase() - для перевода в верхний регистр

строка.trim() - удаление начальных и концевых пробелов в стоке

строка1.concat(строка2) - объединяет две строки

строка.slice(1, 5) - вырезает с 1го по 5й символ с начала (а если с "-" - то с конца)

##МАССИВ

let masiv=[]; - создание массива

массив[1] - второе слово в массиве

массив.length - выводит количество слов в массиве

массив[1] = А - меняет 2й объект на 'А'

массив.toString(разделитель между объектами) - превращает массив в строку (= массив.join() )

Array.isArray(массив) проверяет, является ли массив массивом.

массив.push(слово) - добавляет слово в конец массива

массив.pop() – извлекает элемент из конца,

массив.shift() – извлекает элемент из начала,

массив.unshift(слово) – добавляет элементы в начало.

массив.splice(аргументы) - умеет всё: добавлять, удалять и заменять элементы. Изменяет сам массив.
Его синтаксис: « массив.splice(начатать с индекса Н, удалить столько-то элементов, додавить туда «слово1», «слово…») »

массив.slice(start, end) – создаёт новый массив, копируя в него элементы с позиции start до end (не включая end).

массив.concat(«слово1», «слово…») – возвращает новый массив: копирует все члены текущего массива и добавляет к нему items. Если какой-то из items является массивом, тогда берутся его элементы.

***Для поиска среди элементов:***

массив.indexOf(item) возвращает индекс первого, найденного элемента из массива.

массив.lastIndexOf(item, pos) – ищет item, начиная с позиции pos, и возвращает его индекс или -1, если ничего не найдено.


массив.includes(value) – возвращает true, если в массиве имеется элемент value, в противном случае false.

массив.filter(условия) - фильтрует элементы через функцию и создаёт новый массив со всеми элементами, прошедшими проверку, задаваемую в передаваемой функции. (Пример: 

массив.filter((el, id) => массив.indexOf(el) === id); - убирает дубли из массива )

массив.find(func) – фильтрует элементы через функцию и отдаёт первое/все значения, при прохождении которых через функцию возвращается true.

массив.findIndex() похож на find, но возвращает индекс вместо значения.

***Для перебора элементов:***

массив.forEach(func) – вызывает func для каждого элемента. Ничего не возвращает.

***Для преобразования массива:***

массив.map(func) – создаёт новый массив из результатов вызова func для каждого элемента.

массив.sort(func) – сортирует массив «на месте», а потом возвращает его.

массив.reverse() – «на месте» меняет порядок следования элементов на противоположный и возвращает изменённый массив.

split/join – преобразует строку в массив и обратно.

reduce(func, initial) – вычисляет одно значение на основе всего массива, вызывая func для каждого элемента и передавая промежуточный результат между вызовами.
